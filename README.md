# Kira-Base
> Base system for Kira Linux.

Build scripts, runit init stages, service definitions, and core system configuration that make up the Kira Linux base layer.

## Stack
- Init: runit (PID 1)
- libc: musl
- Userland: BusyBox + util-linux
- Device management: eudev

## Status
Pre-development. See the [Kira Linux specification](https://github.com/shinigami-os) and the project roadmap.
